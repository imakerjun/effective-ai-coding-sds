# 0.3 실습 환경 세팅

이 강의의 미니 실습을 따라가려면 아래 중 **최소 1개**의 환경이 필요합니다. 전부 설치할 필요는 없습니다.

## 선택지

### 옵션 1: Claude Code (권장 — 강의 데모 기준)

Anthropic 공식 CLI. 이 강의의 데모·스크린샷·실습은 기본적으로 Claude Code 기준입니다.

**설치**:
```bash
npm install -g @anthropic-ai/claude-code
claude login
```

**필요 요건**:
- Node.js 18+
- Claude 계정 (Pro/Team/Enterprise 중 하나)

## 자주 막히는 지점

### "로그인했는데 도구 호출이 안 돼요"

- 권한 문제일 수 있음 — CLI가 현재 디렉터리에 읽기/쓰기 권한이 있는지 확인
- Claude Code는 첫 실행 시 폴더 권한 동의를 요청합니다

### "CLAUDE.md를 만들었는데 에이전트가 안 읽는 것 같아요"

- 에이전트를 **CLAUDE.md가 있는 폴더에서 실행**했는지 확인
- 하위 폴더에서 실행하면 상위의 CLAUDE.md를 못 볼 수 있음

### "에이전트가 파일을 못 찾는다고 해요"

- 절대 경로로 명시해보세요
- 또는 `ls`·`pwd` 같은 기본 명령으로 현재 위치를 확인시키세요
