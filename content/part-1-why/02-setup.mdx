# 1.2 실습 환경 세팅

이 강의의 미니 실습은 AI 코드 에이전트 환경 1개만 있으면 됩니다.

## 삼성SDS 수강생 — AI Pro

이 강의는 삼성SDS 사내 도구인 **AI Pro**를 기준으로 설계됐습니다.

### 구성 요소

| 구성 | 설명 |
|---|---|
| **AI Pro IDE 확장** | JetBrains / VSCode / Eclipse 플러그인 — Chat·Agent·Tools·Editor 통합 |
| **AI Pro CLI** | 터미널 도구 (gemini-cli 기반) |
| **`AGENTS.md`** ⭐ | **프로젝트 1차 컨텍스트 파일** — `/init`으로 자동 생성 |
| **Rules** | 행동 지침 (Cursor Rules와 동일 개념) — Personal/Project, AGENTS.md를 보완 |
| **Skills** | 작업별 지침 (Claude의 Skills와 동일 개념) — `/skill-name` 호출 또는 자동 판단 |
| **Pre-set Prompt** | 자주 쓰는 프롬프트 5개까지 저장 |

### 핵심 위치

| 항목 | 경로 |
|---|---|
| **AGENTS.md** ⭐ | `./AGENTS.md` (프로젝트 루트, `/init`으로 생성) |
| Personal Rules | `~/.aipro/rules/*.md` |
| Project Rules | `.aipro/rules/*.md` |
| Personal Skills | `~/.aipro/skills/<skill-name>/SKILL.md` |
| Project Skills | `.aipro/skills/<skill-name>/SKILL.md` |
| CLI 설정 | `~/.gemini/settings.json` |
| Ignore | `.aiproignore`, `.geminiignore` |

**모델 선택**: Qwen3-Coder-128K, GPT-OSS-128K, GLM-4.7-128K 등 (Chat/Agent 우측 상단 드롭다운)

> ⚠️ **Rules 합산 10,000자 제한**: Project Rules + Personal Rules는 합쳐 10,000자까지만 LLM에 전달됩니다. 핵심 규칙을 Project Rules 앞쪽에 두세요.

## 실습 저장소 (공식 안내) ⭐

> 이 강의의 모든 미니 실습은 **하나의 실습 저장소**를 중심으로 진행됩니다.

**저장소**: [imakerjun/autoresearch-harness-steps](https://github.com/imakerjun/autoresearch-harness-steps) — Andrej Karpathy의 [autoresearch](https://github.com/karpathy/autoresearch) fork에 강의 실습 자료를 추가한 형태입니다.

```bash
git clone https://github.com/imakerjun/autoresearch-harness-steps.git
cd autoresearch-harness-steps
```

5가지 토픽에 1:1 매핑되는 5단계가 들어 있습니다:

| Step | 토픽 | 디렉터리 |
|---|---|---|
| 1 | Context | [`harness-steps/step-1-context/`](https://github.com/imakerjun/autoresearch-harness-steps/tree/master/harness-steps/step-1-context) |
| 2 | Plan | [`harness-steps/step-2-plan/`](https://github.com/imakerjun/autoresearch-harness-steps/tree/master/harness-steps/step-2-plan) |
| 3 | Token | [`harness-steps/step-3-token/`](https://github.com/imakerjun/autoresearch-harness-steps/tree/master/harness-steps/step-3-token) |
| 4 | Quality | [`harness-steps/step-4-quality/`](https://github.com/imakerjun/autoresearch-harness-steps/tree/master/harness-steps/step-4-quality) |
| 5 | Multi-Agent | [`harness-steps/step-5-multi-agent/`](https://github.com/imakerjun/autoresearch-harness-steps/tree/master/harness-steps/step-5-multi-agent) |

**왜 빈 스캐폴드가 아니라 실제 오픈소스인가**:
- **실제 코드베이스**에는 컨벤션·역사·제약이 있어, AI 작업의 진짜 어려움을 체감할 수 있음
- Karpathy 본인 프로젝트라 강의 spine과 자연 연결 (부록 F Autoresearch)
- 약 1,000줄 코드 — 한 번에 다 읽을 수 있는 크기
- **`program.md`** — Karpathy가 직접 쓴 *컨텍스트 파일의 교과서적 예시* 가 이미 들어 있음
- **GPU 없이도** 5단계 모두 진행 가능 (코드 *읽기·수정·테스트* 위주)
- `CLAUDE.md` + `AGENTS.md` 모두 포함 — Claude Code · AI Pro 모두 지원
