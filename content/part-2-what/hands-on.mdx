# 🛠️ 실습 트랙 시작하기

Part 2의 5가지 토픽(Context · Plan · Token · Quality · Multi-Agent)을 **하나의 같은 레포를 점진적으로 업그레이드**하면서 손으로 익히는 트랙입니다. 개념 페이지를 이미 봤다면, 이 트랙만 1→5 순서대로 따라가도 됩니다.

## 사전 준비 (5분)

### 1. 실습 저장소 클론

```bash
git clone https://github.com/imakerjun/autoresearch-harness-steps.git
cd autoresearch-harness-steps
```

저장소 안의 `harness-steps/` 디렉터리에 `step-1-context` ~ `step-5-multi-agent` 5개의 폴더가 있습니다. 각 step은 **이전 step의 결과 위에 새 기법 하나**를 얹는 구조입니다.

### 2. 도구 준비

| 도구 | 용도 |
|---|---|
| Claude Code 또는 AI Pro CLI | 5개 step 모두에서 사용 |
| Python 3.11+ | `train.py` 실행용 (step 2~5) |
| 터미널 / 에디터 | 컨텍스트 파일 작성 |

> AI Pro CLI 기준으로 설명하지만, Claude Code도 동일한 흐름으로 따라갈 수 있습니다. 차이가 있는 부분은 각 step에서 별도로 표시합니다.

### 3. 학습 동선

| 순서 | 무엇을 얻는가 | 소요 |
|---|---|---|
| [Step 1 · CLAUDE.md](./hands-on-1-context) | 컨텍스트 파일이 결과를 바꾸는 체험 | 5분 |
| [Step 2 · Plan Mode](./hands-on-2-plan) | "어, 이게 아닌데"를 먼저 차단 | 5분 |

각 step 끝에는 "다음 step으로 →" 링크가 있습니다. 막히면 해당 토픽의 개념 페이지로 돌아가 복습하고 오세요.
