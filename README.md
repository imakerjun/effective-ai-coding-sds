# Effective AI Coding for SDS

멀티캠퍼스 × 삼성SDS 온라인 강의의 강의 문서 사이트.

> **개발자가 AI 코드 에이전트를 "감"이 아닌 원칙으로 다루어,
> 팀 전체가 일관되고 검증 가능한 방식으로 활용할 수 있게 한다.**

## 구성

- 영상 분량: 60~80분 (65분 매핑 기준)
- 프레이밍: **Harness Engineering** (Karpathy, Anthropic)
- 5가지 축: Context / Plan / Token / Quality / Multi-Agent

## 목차

- Part 0. 시작하기 전에
- Part 1. WHY — 우리는 왜 다시 배워야 하는가
- Part 2. WHAT — 하네스 엔지니어링의 5가지 축
- Part 3. 팀에 적용하기
- 부록 (A~F)

## 로컬 실행

```bash
npm install
npm run dev
# http://localhost:3000
```
